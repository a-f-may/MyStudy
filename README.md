## 整数キャッシュとは？

Pythonは `-5` から `256` の整数をあらかじめキャッシュ（インターン）しています。
この範囲の整数は同じオブジェクトを再利用するため、`is` 演算子で比較すると `True` になります。

```python
a = 256
b = 256
print(a is b)  # True（キャッシュ範囲内）

a = 257
b = 257
print(a is b)  # False（キャッシュ範囲外）
```
