while コンパイルエラーパターン（3回目：2026-04-04）

  • ループ外の break / continue は SyntaxError
  • コロン忘れ・インデント誤りも SyntaxError / IndentationError
  • 到達不可コードは Python ではエラーにならないが、論理バグになる
  • ループ変数が変化しない while は無限ループ

Q1

このコードを実行するとどうなりますか？

x = 5
break
解答を見る
SyntaxError: 'break' outside loop

break はループの中でしか使えません。ループ外に書くと構文エラーになります。

Q2

このコードを実行するとどうなりますか？

x = 5
continue
解答を見る
SyntaxError: 'continue' not properly in loop

continue も同様にループ外では使えません。

Q3

このコードを実行するとどうなりますか？

x = 10
while x > 0
    x -= 1
解答を見る
SyntaxError: expected ':'

while 文の条件式の末尾にはコロン : が必要です。

Q4

このコードを実行するとどうなりますか？

while True:
print("hello")
解答を見る
IndentationError: expected an indented block

while ブロックの中身はインデントが必要です。

Q5

このコードはエラーになりますか？なる場合は何のエラーですか？

while True:
    print("hello")
print("終了")
解答を見る

構文エラーにはなりません。ただし while True は無限ループのため print("終了") には永遠に到達しません（到達不可コード）。Python はこれをエラーとして検出しません。

Q6

このコードはエラーになりますか？なる場合は何のエラーですか？

while True:
    break
    print("ここは？")
解答を見る

構文エラーにはなりません。break の直後のコードは到達不可ですが、Python はエラーにしません。print("ここは？") は実行されません。

Q7

このコードはエラーになりますか？なる場合は何のエラーですか？

x = 3
while x > 0:
    print(x)
解答を見る

構文エラーにはなりませんが、無限ループです。x の値がループ内で変化しないため、条件 x > 0 が常に True になり続けます。

修正例：

x = 3
while x > 0:
    print(x)
    x -= 1

Q8（総合）

次の4つのコードについて、エラーの種類（SyntaxError / IndentationError / エラーなし）を答えてください。

# (a)
for i in range(3):
    pass
break

# (b)
while False:
    print("hello")

# (c)
while True:
        print("hello")

# (d)
x = 0
while x < 3:
    x += 1
continue
解答を見る
  • (a) SyntaxError — break がループ外
  • (b) エラーなし — 条件が最初から False なのでループ本体は実行されないだけ
  • (c) IndentationError — インデントが一貫していない
  • (d) SyntaxError — continue がループ外
