# 2026-03-29：基礎問題 — `is` 演算子と整数キャッシュの基本

## 学習目標

- Pythonの整数キャッシュ範囲（`-5` 〜 `256`）を覚える
- `==`（値の比較）と `is`（同一オブジェクトの比較）の違いを理解する
- キャッシュ範囲内・範囲外での `is` の挙動を予測できる

---

## 問題 1

次のコードを実行すると何が表示されますか？

```python
a = 100
b = 100
print(a is b)
print(a == b)
```

<details>
<summary>解答を見る</summary>

```
True
True
```

**解説：** `100` はキャッシュ範囲（-5〜256）内なので、`a` と `b` は同じオブジェクトを指します。`is` も `==` も `True` になります。

</details>

---

## 問題 2

次のコードを実行すると何が表示されますか？

```python
a = 257
b = 257
print(a is b)
print(a == b)
```

<details>
<summary>解答を見る</summary>

```
False
True
```

**解説：** `257` はキャッシュ範囲外なので、`a` と `b` は別々のオブジェクトとして生成されます。値は等しいので `==` は `True`、でも別オブジェクトなので `is` は `False` です。

</details>

---

## 問題 3

次のコードを実行すると何が表示されますか？

```python
a = -5
b = -5
print(a is b)

x = -6
y = -6
print(x is y)
```

<details>
<summary>解答を見る</summary>

```
True
False
```

**解説：** `-5` はキャッシュ範囲の下限なので `is` は `True`。`-6` は範囲外なので別オブジェクトとなり `False` になります。

</details>

---

## 問題 4

次のコードを実行すると何が表示されますか？

```python
a = 256
b = 256
c = 257
d = 257

print(a is b)
print(c is d)
print(a is not b)
print(c is not d)
```

<details>
<summary>解答を見る</summary>

```
True
False
False
True
```

**解説：** `256` はキャッシュ範囲の上限（境界値）です。`257` は範囲外です。`is not` は `is` の否定です。

</details>
